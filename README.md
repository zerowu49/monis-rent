# Monis Rent — Interactive Workspace Designer

Design your dream workspace in interactive room with windows overlooking lush rice paddy fields. Pick a desk, choose a chair, add accessories, and see your setup come to life — then hit "Rent" to make it real.

## Capabilities

- Visual builder with live preview in room (rice paddy window views)
- Users could choose a desk, chair, or accessories (monitors, keyboard, webcam, etc.)
- User could simulate renting the workspace & price projection
- Responsive, smooth animations, and tropical themed UI

## Tech Stack

- React + TypeScript
- Next JS
- Tailwind CSS

## Quick Start

- Install: npm install
- Develop: npm run dev
- Build: npm run build

## Project Structure (Key Files)

- src/app/page.tsx — Main app and layout
- src/components/ — Canvas, selectors, and summary components
- src/data/products.ts — Catalog data for desks, chairs, and accessories
